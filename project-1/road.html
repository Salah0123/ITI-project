Front End

UI UX

تعلمك لل ui ux سيجعلك قادرا على تقديم افضل تجربة للمستخدم من ناحية التصميم, قد يعتقد الكثيرون ان ال ui ux غير مهم او ان الشركات تستطيع الاستغناء عنها , لكن على العكس تماما ف ui ux يعد من اهم الوظائف و فبدونه لن تعرف ما هو التصميم الافضل للمستخدمين و وستكون غير قادرا على تقديم افضل تجربة للمستخدمز

HTML

يمكنك اعتبار ال HTML هو الخطوة الاولى الفعلية في بناء موقع الويب الخاص بك و فمن خلال ال HTML تستطيع تصميم هيكل الموقع وبدونها لا يمكنك دخول هذا المجال

CSS

بعد ما قمت بتصميم هيكل الموقع بال HTML .. يجب عليك تنسيق هذا الهيكل وترتيبه واللغة التي تفعل هذا هي CSS .. تستخدم CSS لعمل تنسيق لموقعك وضبط ابعاده وعمل تأثيرات مختلفة واضافة صور والوان وجعل شكله رائع

Java Script

تعتبر اهم خطوة لك فى الفرونت اند وتعتبر اول لغة برمجة ستتعامل معها ويجب ان تكون على دراية بال BASICS – DOM – BOM – OOP – API

JS Framework

للتسهيل عليك ولتقديم افضل تجربة باقل وقت فيجب عليك استخدام فريم ورك للجافاسكريبت يمكنك الاختيار بين العديد منها مثال Vue.js – React.js – Angular.js

Programing Language

في هذه الخطوة يمكنك اختيار اي لغة برمجة تدعم التعامل مع الويب ويوجد الكثير مثال PHP – Python وغيرها من اللغات كل ما عليك هو اختيار لغة برمجة واحدة والعمل بها وللتسهيل عليك فالاغلب يختار PHP كونها من اكثر لغات البرمجة في العالم التي صنع بها موقع الويب

Framework

بعد اختيارك للغة البرمجة التي ستعمل بها فى الباك اند .. فالخطوة التالية لك هو اختيار بيئة عمل للتسهيل عليك وتقديم افضل كود فى اقل وقت ممكن .. وكل لغة برمجة ولها بيئة عمل مناسبة ف اذا كان الاختيار لل PHP فبيئة العمل المناسبة هي Laravel .. واذا كان اختيارك للغة بايثون ف ان بية العمل المناسبة هي Django – Flask

My SQL

بعد تعلمك كل هذا يجب ان تتعامل مع قواعد بيانات لحفظ بيانات مشروعك بها وقواعد بيانات My SQL من اشهر قواعد البيانات فى عالم الويب وهي المسؤولة عن التعامل مع قاعدة البيانات الخاصة بتطبيقك او مشروعك وارسال واستقبال البياانات منها

Publish your site

بعد اتمامك لموقع الويب يجب عليك نشره على شبكة الانترنت العالمية لكي يراه جميع المستخدمين ولجعل ملفات موقع مرئية للعامة يجب عليك رفعها على استضافة .. ولفعل هذا يجب عليك اختيار شركة استضافة مناسبة لك ويجب عليك ان تعلم ان خدمة الاستضافة فى الغالب غير مجانية فستقوم بشراء استضافة من احد الشركات التي تعطي مساحة تخزين لحفظ ملفات الويب مثال Hostinger - Bluehost
